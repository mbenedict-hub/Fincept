# Fincept Website Prototype

This repository contains a static financial analysis website prototype and a GitHub Pages workflow for automatic deployment.

## Live site

After GitHub Pages finishes deploying, the site will be available at:

https://mbenedict-hub.github.io/Fincept/

## Deployment

This repository deploys automatically with GitHub Actions whenever changes are pushed to the main branch.

GitHub Pages serves the static site from the workflow artifact.
